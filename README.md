# Prontofy Links

Projeto de links e direcionamentos da Prontofy, desenvolvido pela REAX Corps.

## Como rodar localmente

Tenha Node.js e npm instalados. Depois rode:

```sh
npm i
npm run dev
```

## Links de direcionamento

Os destinos dos botĂ”es ficam em `src/config/links.ts`.

Quando as URLs finais estiverem definidas, atualize:

- `plataforma`
- `iaWhatsapp`
- `recursos`
- `telemedicina`

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
